Frequently Asked Questions

Q1. How can I find a nearby rental or returning point?

Make sure GPS is enabled while using the “HK Easy Charge App” or “WeChat Mini Programs”. Users can use the “Search” tab to find a specific location or select any nearby location on the map to rent/return a power bank.

Q2. How many power bank can be rented simultaneously?

"Single Rental" user can rent more than one power bank provided the deposit is paid for each power bank. For VIP members, please refer to point 6.
